时间格式参照 java 时间格式规范:<br/>
yyyy 代表年,如果位数不够,从后向前取值,比如 2019 年,使用 yy 就是 19<br/>
MM 代表月,MMM 代表中文月 11 月的话,MM 就是 11,MMM 就是十一月<br/>
dd 代表日<br/>
HH 代表小时,24 小时制<br/>
mm 代表分钟<br/>
ss 代表秒<br/>
农历年 代表农历的年份(如:乙亥猪年)<br/>
农历月 代表农历的月份(如:正月初一)<br/>
aaa 代表上午或者下午<br/>
EEEE 代表星期几<br/>
默认是:yyyy 年 MMMdd 日 EEEE HH:mm
